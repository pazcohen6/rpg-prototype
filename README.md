# Project Umbra

A 2D turn-based RPG made in Godot where the player survives enemy encounters using abilities.

For more details, see the [Game Design Document](GDD.md).

## Built With

- Godot Engine v4.5.1
- GDScript

## Project Status

This project is currently a learning prototype created to improve my skills in:
- Godot Engine
- Programming
- GDScript
- Game architecture and game design
- Turn-based combat design

## Project Goals

- Implement a functional turn-based combat system
- Implement a stats system
- Implement an inventory and equipment system
- Implement a functional skill tree

## How to Run

1. Download Godot Engine v4.5.1
2. Clone the repository
3. Open the project in Godot
4. Run the main scene

## Controls

- Point and click
